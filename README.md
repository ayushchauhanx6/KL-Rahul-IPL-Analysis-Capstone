#KL Rahul: IPL Batting Evolution Analysis (2025-2026)
## Project Overview
This data analytics project tracks the statistical evolution of cricket player KL Rahul following his transition to the Delhi Capitals.
## Data Stack Used
* **SQL (Google Cloud Bigquery) :**Data extraction,merging and strike rate calculations.
* **Python:** Basic Data processing and automation.
* **Tableau:** Visualizing seasinal trendlines (Runs vs Strike Rate)
